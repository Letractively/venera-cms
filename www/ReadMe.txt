Установка
_________________________________________________________________


1. Установите права на запись для файлов:
app/install.txt
app/settings/sites/default/default.php
app/settings/sites/default/admin.php


2. Откройте страницу установки - http://yoursite/install/

3. Проверьте работоспособность сайта

4. Если файл install.txt не удалён, удалите его
